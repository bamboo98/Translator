# Vosk模型目录

请将下载的Vosk语音识别模型放在此目录下。

## 模型下载

访问 [Vosk模型下载页面](https://alphacephei.com/vosk/models) 下载所需语言的模型。

## 推荐模型(如果内存不足则使用较小的模型)

- **日文**: `vosk-model-ja-0.22`
- **英文**: `vosk-model-en-us-0.22`

## 目录结构示例

```
models/
├── vosk-model-ja-0.22/
│   ├── am/
│   ├── graph/
│   └── ...
├── vosk-model-en-us-0.22/
│   └── ...
└── ...
```

## 注意事项

1. 下载后解压模型文件
2. 将解压后的文件夹（不是文件）放到此目录
